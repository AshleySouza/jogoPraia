# 🏖️ Jogo da Praia - Escorpiões

Um jogo estilo Chrome offline com tema de praia.

## 🎮 Como Jogar

- **ESPAÇO** ou **SETA PARA CIMA** → Pular
- Evite os escorpiões!

## 👧 Personagem

- Mulher com cabelo branco comprido
- Pele clara
- Camiseta rosa com escrito "LOVE"
- Calça cargo cinza

## 🌅 Fases

| Tempo | Modo |
|-------|------|
| 0-60s | Praia de dia |
| 60s+ | Praia de noite |

- A cada 30 segundos a velocidade aumenta
- Aos 4 minutos toca um som especial

## 🚀 Como Subir para o GitHub

```bash
# 1. Criar repositório no GitHub
# Acesse github.com e crie um novo repositório

# 2. No terminal, na pasta do jogo:
git init
git add .
git commit -m "Primeiro commit - Jogo da Praia"

# 3. Conectar com seu repositório
git remote add origin https://github.com/SEU_USUARIO/jogo-praia.git

# 4. Enviar para o GitHub
git push -u origin main
```

## 🌐 Como Jogar Online

Após subir para o GitHub:
1. Acesse seu repositório
2. Vá em **Settings** → **Pages**
3. Em **Branch** selecione `main` e salve
4. O jogo estará disponível em `https://SEU_USUARIO.github.io/jogo-praia/`
